# Security Policy

## 数据安全

本项目涉及考生个人信息（姓名、分数、位次、偏好等），属于敏感个人数据。所有任务产物仅在本地 `output/` 目录生成，**不要**将真实考生资料提交到版本控制系统或公共平台。

## 脚本安全

项目脚本仅使用 Python 标准库，不会主动联网、安装依赖或执行外部命令。`render_volunteer_draft.py` 生成的 `.xlsx` 文件是标准 Office Open XML 格式，不包含宏。

## 报告安全问题

如果你发现脚本存在以下安全隐患，请通过 GitHub Issue 或私下联系维护者，并附上复现步骤：

- 脚本可能执行非预期命令或读取无关文件；
- 生成的文件中泄露本地绝对路径或个人隐私信息；
- 数据查询逻辑可能导致错误的填报建议。

## 不应提交的内容

- API key、token、cookie、密码；
- 真实考生姓名、分数、位次、身份证号等个人信息；
- 本地绝对路径、个人隐私信息；
- `output/` 目录下的运行产物。
